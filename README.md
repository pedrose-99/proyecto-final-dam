# SmartCart

Comparador de precios de supermercados.

## Requisitos

- Docker
- Docker Compose

## Ejecutar

```bash
docker-compose up --build 
o
docker-compose up --build -d

```

## URLs

| Servicio | URL |
|----------|-----|
| Frontend | http://localhost:4200 |
| Backend | http://localhost:8081 |
| PostgreSQL | localhost:5434 |

## Parar

```bash
docker-compose down
```
